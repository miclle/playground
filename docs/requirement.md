# Playground - AI Coding Desktop App

基于 Electron.js 构建的桌面应用，通过 AI 在安全沙箱中生成和执行代码，实现 vibe coding 人机协作开发。

## 产品特性

- **开箱即用**：应用独立运行，无需宿主机安装 Node.js、Python 等运行时环境
- **安全隔离**：AI 生成和执行的代码在云端沙箱中运行，不影响宿主机
- **人机协作**：支持自然语言驱动 AI 编程，也支持用户直接编辑代码

## 核心功能

### 1. AI 代码生成

- 通过自然语言对话驱动代码生成（vibe coding）
- 支持用户直接编辑代码，实现人机协作
- AI 可在 Sandbox 中执行完整操作：生成代码、运行命令、安装依赖、读写文件
- 支持基于已有代码的多轮迭代修改

**AI 服务支持：**
- OpenAI API（GPT / Codex）
- Claude API（Claude）

### 2. Sandbox 沙箱

所有 AI 操作在云端沙箱中隔离执行，API 协议兼容 [e2b](https://e2b.dev/docs)。

**工作原理：**
- 沙箱运行在云端服务器（e2b 或兼容服务），不在本地执行
- 桌面应用通过 API 与沙箱通信，发送指令和接收结果
- 用户无需在本地安装任何运行时环境

**沙箱特性：**
- **生命周期**：按会话管理，用户可手动暂停/恢复/销毁
- **环境模板**：预置 Node.js、Python、Go 等运行时环境
- **文件系统**：会话内持久化，会话结束后由用户选择保存或丢弃
- **网络**：默认允许出站访问（安装依赖等），支持代理配置

### 3. Artifact 产物管理

AI 在 Sandbox 中生成的所有产物支持保存到多种存储后端：

- **本地文件系统**
- **对象存储**（AWS S3 等）
- **GitHub**（push 到已有仓库或创建新仓库）
- **云盘 / 数据库**

产物按项目组织，支持版本记录和历史查看。

### 4. 用户界面

采用三栏布局：

- **左侧**：文件树，浏览 Sandbox 中的项目结构
- **中间**：代码编辑器（Monaco Editor），支持语法高亮和直接编辑
- **右侧**：AI 对话面板，自然语言交互
- **底部/可切换**：终端输出、实时预览（WebView 渲染前端页面）

### 5. 项目与会话

- 以"项目"为单位组织代码和对话，一个项目对应一个 Sandbox 实例
- 对话历史持久化保存，支持查看和继续
- 单用户使用，不涉及多人协作

## 配置管理

用户可自定义配置以下服务：

| 配置项 | 内容 |
|--------|------|
| AI 服务 | API URL、API Key、模型选择 |
| Sandbox | 服务地址、API Key、环境模板 |
| 存储后端 | 各存储类型的连接信息和认证凭据 |

- API Key 等敏感信息使用系统 Keychain 安全存储
- 支持多套配置 profile 切换

## 平台支持

- macOS（优先）
- Windows、Linux（后续支持）

## 分发方式

- 打包为独立可执行文件，内置 Node.js 运行时
- 用户无需安装任何依赖即可运行
- 支持自动更新
