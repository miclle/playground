# Playground - AI Coding Desktop App

基于 Electron.js 构建的桌面应用，通过 AI 在安全沙箱中生成和执行代码，实现 vibe coding 人机协作开发。

## 核心功能

### 1. AI 代码生成

- 通过自然语言对话驱动代码生成（vibe coding）
- 支持用户直接编辑代码，实现人机协作
- AI 可在 Sandbox 中执行完整操作：生成代码、运行命令、安装依赖、读写文件
- 支持基于已有代码的多轮迭代修改

**AI 服务支持：**
- OpenAI API（Codex）
- Claude API（Claude Code CLI）

### 2. Sandbox 沙箱

所有 AI 操作在沙箱中隔离执行，API 协议兼容 [e2b](https://e2b.dev/docs)。

- **生命周期**：按会话管理，用户可手动暂停/恢复/销毁
- **环境**：支持主流语言运行时（Node.js、Python、Go 等），由沙箱模板定义
- **文件系统**：会话内持久化，会话结束后由用户选择保存或丢弃
- **网络**：默认允许出站访问（安装依赖等），支持代理配置

### 3. Artifact 产物管理

AI 在 Sandbox 中生成的所有产物支持保存到多种存储后端：

- **本地文件系统**
- **对象存储**（AWS S3 等）
- **GitHub**（push 到已有仓库或创建新仓库）
- **云盘 / 数据库**

产物按项目组织，支持版本记录和历史查看。

### 4. 用户界面

采用三栏布局：

- **左侧**：文件树，浏览 Sandbox 中的项目结构
- **中间**：代码编辑器（Monaco Editor），支持语法高亮和直接编辑
- **右侧**：AI 对话面板，自然语言交互
- **底部/可切换**：终端输出、实时预览（WebView 渲染前端页面）

### 5. 项目与会话

- 以"项目"为单位组织代码和对话，一个项目对应一个 Sandbox 实例
- 对话历史持久化保存，支持查看和继续
- 单用户使用，不涉及多人协作

## 配置管理

用户可自定义配置以下服务：

| 配置项 | 内容 |
|--------|------|
| AI 服务 | API URL、API Key、模型选择 |
| Sandbox | 服务地址、API Key、环境模板 |
| 存储后端 | 各存储类型的连接信息和认证凭据 |

- API Key 等敏感信息使用系统 Keychain 安全存储
- 支持多套配置 profile 切换

## 平台支持

- macOS（优先）
- Windows、Linux（后续支持）
